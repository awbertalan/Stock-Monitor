Stock Page Auto-Refresh Interval
How often the chart refreshes automatically when viewing a stock page. Set to Manual only to disable auto-refresh entirely. Takes effect on the next page load.
Trades List Limit
How many recent trades to display on the stock page (10–50). Takes effect on the next stock page load.
Update All Scope
The Update All button in the navigation bar only refreshes stocks whose instrument type and currency are both checked below.
The Update Everything button at the bottom always refreshes all stocks, regardless of this filter.

Update ISIN numbers for all stocks in the database. This will fetch and populate missing ISIN codes from Millistream. May take a minute or two.
Market List Database
Populate the market list (OMXS30, Large Cap, Mid Cap, Small Cap, First North, Spotlight) for every stock. Required for the dashboard list tabs to show data. Only fetches stocks with a missing list — safe to re-run.
